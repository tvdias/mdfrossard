## 🔝 The Golden Rules (Antigravity/Gemini Optimized)

1. **Local Verification**: `npm run build` MUST succeed after every change. Testing on `http://localhost:4000`.
2. **Local Assets**: All images MUST be saved locally in `source/l/<page>/img/`. **NO external CDN links.**
3. **Image Perf**: Use `<picture>` with **WebP** source and **JPG** fallback. Always.
4. **Landing Pages**: Folder `source/l/`, Base `source/l/exemplo/`, Layout `layout_landing`.

---

## 📂 Quick Project Layout
```text
/source/_posts/          ← Blog content (pt_BR)
/source/l/               ← Professional Landing Pages
/themes/mdfrossard/      ← Layouts and Partials (.ejs)
/source/_data/           ← Global configs and content data
.eleventy.js             ← Generator settings (passthrough rules)
```

---

## ⚡ Main Scripts
- `npm start`: Runs `eleventy --serve` (port 4000).
- `npm run build`: Generates site into `public/`.

---

## 🚨 Final Reminder
- **Language**: Always Brazilian Portuguese (`pt_BR`).
- **SEO**: Every new page needs a unique `title` and `description` in frontmatter.
- **Mime Headers**: Ensure downloaded WebPs are true WebPs (RIFF header).
