## Quick Visual Summary

### The Golden Rule
After EVERY change:
```bash
npm run build              # Must succeed
npm start:app              # Must start
# Test at http://localhost:4000
```

### Key Commands at a Glance
| Purpose | Command |
|---------|---------|
| Full build | `npm run build` |
| Build site only | `npm run build:app` |
| Run site | `npm start:app` |
| Clean cache | `npx hexo clean` |

### Project Layout
```
source/_posts/          ← Blog posts (edit here)
source/_drafts/         ← Unpublished drafts
themes/mdfrossard/      ← Theme (careful!)
functions/              ← Netlify functions
public/                 ← GENERATED (don't edit)
_config.yml             ← Site config (careful!)
```

### Critical Rules
1. 🚨 Verify build/runtime after EVERY change
2. ✅ Commit with clear messages
3. 🌍 Use Portuguese (pt_BR) for content
4. 🚫 Never edit public/ directory
5. 📝 Keep YAML syntax valid
6. 🔗 Test links after modifications

### Git Workflow
```bash
git checkout -b feature/description
# Make changes
npm run build && npm start:app          # Verify
git add . && git commit -m "Description"
git push origin feature/description
```
